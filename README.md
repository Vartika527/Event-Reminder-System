# Event Reminder System (Java)

A simple console-based Event Reminder System built using Java.

## Features
- Add new events
- View all events
- Mark events as completed
- Remove events
- Show completed events
- Date validation using LocalDate

## Technologies Used
- Java
- OOP Concepts
- ArrayList
- LocalDate API

## How to Run
1. Compile:
   javac Executer.java

2. Run:
   java Executer
